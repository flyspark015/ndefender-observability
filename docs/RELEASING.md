## Prereqs
- Tag pushed: `v0.1.0-observability-green`

## GitHub UI (Manual)
1. Open the GitHub repo Releases page.
2. Click "Draft a new release".
3. Tag: `v0.1.0-observability-green`
4. Title: `v0.1.0 Observability (GREEN)`
5. Paste notes from `/tmp/release_notes.md` (generated by tool below).
6. Publish.

## Generate Release Notes
```bash
python3 tools/make_release_notes.py
```

## GitHub CLI (Optional)
If `gh` is installed and authenticated:
```bash
python3 tools/make_release_notes.py

gh release create v0.1.0-observability-green \
  --title "v0.1.0 Observability (GREEN)" \
  --notes-file /tmp/release_notes.md

gh release view v0.1.0-observability-green
```
